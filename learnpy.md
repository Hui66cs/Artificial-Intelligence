## 索引
1. [-1]  # 获取最后一个元素
2. [:] # 获取中所有元素，返回一个对应类型的对象（如列表，元组）
3. [a:b]  # 获取索引a到b-1的元素，返回一个新列表(左闭右开区间)


## list

### 声明
li = []
other_li = [4, 5, 6]

### 操作
1. 添加元素： li.append(1)  # 在列表末尾添加元素
2. 插入元素： li.insert(2, 3)  # 在索引2处插入元素3
3. 弹出元素： li.pop()  # 弹出列表末尾的元素，返回该值
4. 删除元素： li.remove(1)  # 删除列表中第一个出现的元素1
5. 删除索引元素： del li[2]  # 删除索引2处的元素
6. 获取元素索引： li.index(3)  # 返回列表中第一个出现的元素3的索引
7. 获取长度： len(li)  # 返回列表的长度
8. 访问元素： li[3]  # 访问列表中的第四个元素，可修改
9. 两个列表合并： new_li = li + other_li  # 合并两个列表
10. 检查存在： 3 in li  # 检查元素3是否在列表li中
11. 获取切片： li[1:4]  # 获取列表中索引1到3的元素，返回一个新列表


## tuple
**不可修改，不可删除元素**

### 声明
tup = (1, 2, 3)

### 操作
1. 获取元素： tup[0]  # 获取元组中的第一个元素
   注意：元组中的元素不可修改，tup[0] = 4  会引发错误
2. 获取长度： len(tup)  # 返回元组的长度
3. 检查存在： 2 in tup  # 检查元素2是否在元组tup中
4. 获取切片： tup[1:3]  # 获取元组中索引1到2的元素，返回一个新元组
5. 两个元组合并： new_tup = tup + (4, 5)  # 合并两个元组，返回一个新元组
6. 元组解包： a, b, c = tup  # 将元组中的元素分别赋值给变量a、b、c


## set
**无序且不重复**

### 声明
s = {1, 2, 3}
s = set()  
s = set([1, 2, 3])  # 从列表创建集合
s = set((1, 2, 3))  # 从元组创建集合

### 操作
1. 添加元素： s.add(4)  # 向集合s中添加元素4
2. 删除元素： s.remove(2)  # 从集合s中删除元素2
3. 清空集合： s.clear()  # 清空集合s中的所有元素



## dict
**键值对，键唯一**

### 声明
d = {'a': 1, 'b': 2, 'c': 3}
d = dict()

### 操作
1. 添加/修改键值对： d['a'] = 4  # 添加新的键值对，或修改已有键的值
2. 删除键值对： del d['b']  # 删除键'b'及其对应的值
3. 获取值： d['c']  # 获取键'c'对应的值
4. 获取所有键： d.keys()  # 返回一个包含所有键的视图
5. 获取所有值： d.values()  # 返回一个包含所有值的视图
